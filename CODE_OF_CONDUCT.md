# Código de Conduta

A comunidade Tunnara deve manter comunicação respeitosa, técnica e colaborativa. Não são aceitos assédio, discriminação, ataques pessoais, exposição de dados privados ou uso do projeto para atividades ilegais.

Incidentes devem ser informados de forma privada pelo canal descrito em `SECURITY.md`. A manutenção pode moderar, editar ou remover conteúdos e restringir a participação quando necessário para proteger a comunidade e o projeto.
