# Tunnara Mobile 2.0.0-rc.4

Clientes móveis para acesso às redes privadas Tunnara.

- `android/`: Kotlin, `VpnService` e WireGuard userspace;
- `ios/`: SwiftUI, `NEPacketTunnelProvider` e WireGuardKit.

Os builds foram separados da publicação nas lojas. Consulte [`docs/mobile/BUILD_AND_DISTRIBUTION.md`](../../docs/mobile/BUILD_AND_DISTRIBUTION.md).
