# Tunnara Platform 2.0.0-rc.2

Release Candidate de hardening operacional e de distribuição.

## Destaques

- pipeline de release imutável e coordenado por release ID;
- correções cross-platform de SEA, Tauri, Android e iOS;
- assets exclusivos e upload idempotente;
- build number mobile monotônico;
- Docker distribuído com fallback TCP e overlay QUIC explícito;
- backup, restore, update e rollback PostgreSQL;
- Dependabot agrupado sem majors automáticos incompatíveis;
- documentação de produção, upgrade, DR e incidentes.

## Compatibilidade

A compatibilidade com ngrok e Pangolin é funcional, não binária nem de protocolo. Esta RC mantém o runtime Node como caminho oficial e o plano Rust como evolução controlada.

## Upgrade

Leia `docs/operations/UPGRADE_ROLLBACK.md`. Uma versão publicada não deve ser substituída; publique sempre uma nova tag.
