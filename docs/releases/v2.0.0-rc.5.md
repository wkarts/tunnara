# Tunnara 2.0.0-rc.5

Release Candidate corretiva para a matriz de builds nativos e mobile.

## Corrigido

- reqwest 0.13 com TLS Rustls;
- dependências criptográficas compatíveis no Console Tauri;
- Gradle 9.4.1 para AGP 9.2.1;
- WireGuardKit local preparado antes do XcodeGen;
- bridge Go externo integrado ao projeto iOS;
- imagem quic-bridge e workflows Docker atualizados;
- validações Cargo adicionadas ao Pull Request.

## Classificação

Release Candidate para homologação. A promoção para GA continua condicionada aos gates documentados em `docs/security/MATURITY_GATES.md`.
