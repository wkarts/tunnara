# Tunnara 2.0.0-rc.1

## Destaques

- Policy Engine no Edge.
- Request Inspector com replay e redação.
- múltiplos targets, health checks e failover;
- Control API distribuído sobre PostgreSQL/Redis;
- autenticação de Agent no Control Laravel;
- nodes, presença e rotas internas persistidas;
- Prometheus e Grafana;
- Helm Chart;
- OpenAPI 2.0 RC;
- fuzzing e benchmark local;
- manutenção automática de retenção.

## Compatibilidade

O runtime embedded continua compatível com instalações SQLite anteriores. O plano distribuído usa migrações Laravel próprias e deve ser implantado em uma nova base ou por processo de migração controlado.

## Classificação

Release Candidate para homologação e produção controlada. Não é uma certificação de escala global ou segurança externa.
