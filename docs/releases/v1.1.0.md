# Tunnara Platform 1.1.0

Release focada em implantação real, previsibilidade de builds e operação simplificada.

## Destaques

- Bundle Docker instalável diretamente pela GitHub Release.
- Quickstart por imagens GHCR ou build local.
- Produção com Cloudflare, Let’s Encrypt DNS-01, HTTP/3 e QUIC.
- Perfis documentados para SQLite/memory e Control API com PostgreSQL/MySQL/Redis.
- Release coordenada em draft até a conclusão de Runtime, SDK, Desktop, Mobile e Containers.
- Imagens GHCR para Server, Agent, Console, Control API, QUIC Bridge e Caddy Cloudflare.
- Console com lazy loading, minificação e divisão de chunks.
- Instalação npm substancialmente reduzida pela remoção de dependências de release não utilizadas.

## Instalação Docker

```bash
git clone https://github.com/wkarts/tunnara.git
cd tunnara
./docker.sh quickstart
```

Ou use o asset `Tunnara-Docker-v1.1.0.zip` disponível nesta release.

## Atualização

```bash
cd deploy/docker
./tunnara.sh update
```

Com SQLite, um backup consistente é criado antes da atualização.

## Observações

- APK/AAB e IPA são gerados sem exigir publicação nas lojas.
- Assinaturas e publicações comerciais continuam opcionais e condicionadas aos secrets do proprietário.
- A stack Community usa SQLite/memory; PostgreSQL/MySQL/Redis pertencem ao plano de gestão Laravel distribuído.
