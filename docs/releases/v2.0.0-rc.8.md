# Tunnara Platform 2.0.0-rc.8

Release Candidate de correção focada em dois pontos do pipeline pós-merge:

- upload idempotente de assets da GitHub Release diante de corridas HTTP 422;
- separação do alvo iOS Simulator do Packet Tunnel e do bridge Go destinado ao device.

A aplicação device e o IPA continuam utilizando WireGuardKit, Packet Tunnel
Extension e WireGuardGoBridgeiOS. O alvo de simulador é exclusivamente um smoke
test de UI e gerenciamento `NetworkExtension`.
