# Tunnara Platform 2.0.0-rc.7

Correção coordenada da identificação e publicação de GitHub Release drafts.

- Draft criada e retomada diretamente pela API REST.
- `release_id` numérico propagado para Core, Runtime, SDK, Desktop e Mobile.
- Upload direto para `uploads.github.com` pelo ID da draft.
- Exclusão idempotente de assets existentes pelo mesmo `release_id`.
- Publicação final por ID, sem depender do endpoint `/releases/tags/{tag}`.
- Teste funcional impede regressão para resolução de drafts por tag.
