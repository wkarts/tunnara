# Branding Tunnara

A identidade visual oficial está centralizada em `src/assets/branding`.

## Assets aplicados

- `logo-light.png`: logotipo horizontal para fundos escuros.
- `logo-dark.png`: derivação para superfícies claras.
- `logo-mark.png`: símbolo isolado e transparente.
- `splash-logo.png`: composição vertical da tela de inicialização.
- `favicon.ico` e `favicon.png`: navegador e atalhos.
- `app-icon-*`: ícones rasterizados de 16 a 1024 pixels.
- `brand-preview.jpg`: imagem de apresentação do repositório.

## Cores oficiais

- Azul primário: `#2F6FED`
- Azul de interação: `#2457C5`
- Cyan de conectividade: `#12C6D4`
- Azul-marinho estrutural: `#071426`
- Superfície suave: `#E7F1FF`

## Identidade técnica

- Produto: `Tunnara Platform`
- Console: `Tunnara Console`
- Bundle ID: `br.com.wwsoftwares.tunnara.console`
- Prefixo de ambiente: `TUNNARA_`
- Binário desktop: `tunnara_console`
- Agente: `tunnara-agent`

Os arquivos fornecidos no pacote `Tunnara_Branding.zip` foram preservados como fonte visual. As derivações de tamanho e a versão para superfícies claras são geradas sem alterar a forma do símbolo.
